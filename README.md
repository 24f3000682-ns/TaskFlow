# 🚀 TaskFlow - Task Management Dashboard

A modern and responsive Task Management Dashboard built using HTML, CSS, and JavaScript.

---

## 📌 Features

- 🔐 User Login Authentication
- ➕ Add New Task
- ✏️ Edit Existing Task
- 🗑️ Delete Task
- ✅ Mark Task as Completed
- 🔍 Search Tasks
- 🎯 Filter Tasks (All / Pending / Completed)
- 📊 Task Progress Bar
- 🌙 Dark / Light Mode
- 🔔 Toast Notifications
- 💾 Local Storage Support
- 📱 Responsive Dashboard Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
